# 山瀬造園 営業提案用デモサイト

職人・建設業者向けの営業DMで先に見せるための、1ページ型静的デモサイトです。本番サイトではありません。

## ローカル確認

依存関係はありません。`public` ディレクトリを静的サーバーで開きます。

```bash
cd public
python3 -m http.server 4173
```

`http://localhost:4173` を開いて確認します。

## Cloudflare Pages

GitHubリポジトリ連携で以下を指定してください。

- Framework preset: `None`
- Build command: 空欄
- Build output directory: `public`
- Root directory: `/`（変更しない）

`public/index.html` が公開されます。`meta name="robots" content="noindex, nofollow"` は実装済みです。

## 主な編集箇所

- 文章・連絡先・リンク: `index.html`
- 配色・余白・レスポンシブ表示: `styles.css`
- 画像: `public/assets/`
- `施工イメージ` のラベル: `index.html` 内の各画像エリア
